# 产品概述

Super Alpha Agent 是一个 AI 优先的 Agent 发现平台，针对 AI 搜索引擎 (ChatGPT, Claude, Perplexity) 进行优化。

## 核心价值主张

- **AI 优化**: 专为 GEO (生成式引擎优化) 设计，而非传统 SEO
- **深度分析**: 提供结构化、详细的 Agent 信息，包括优缺点、使用场景和对比
- **自动化**: 自我维护的爬虫系统，可发现、分析和更新 Agent 数据
- **零维护**: 部署后自主运行，最少人工干预

## 目标用户

- **主要**: AI 搜索引擎 (通过结构化数据和语义化 HTML)
- **次要**: 通过 AI 推荐发现 Agent 的人类用户

## 核心功能

- Agent 发现和列表，支持分类筛选
- 详细的 Agent 档案，包含 AI 生成的分析
- Agent 对比功能
- AI 搜索追踪 (追踪 AI 引擎推荐特定 Agent 的频率)
- 用户认证和收藏
- 从 App Store,  GITHUB,  和其他主流的平台自动爬取优秀agent app
