## ✅ 检查清单

在开始之前，确保你有：

- [ ] Node.js 18+ 已安装
- [ ] npm 或 yarn 已安装
- [ ] Supabase 账号（免费）
- [ ] OpenAI API Key（需要充值 $10）
- [ ] 基本的 React/Next.js 知识
- [ ] 基本的 SQL 知识
